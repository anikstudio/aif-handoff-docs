01
Системные требования
Что нужно установить перед началом
⬢
Node.js 22+
Рекомендуется установка через nvm (Node Version Manager) — позволяет легко переключать версии.
🤖
Claude Code CLI
Agent SDK запускает Claude Code как дочерний процесс — CLI обязателен. Устанавливается через npm install -g или нативный инсталлятор.
🔑
Claude аккаунт
Подписка Claude (OAuth через claude login) или Anthropic API Key. Credentials хранятся в ~/.claude/.
🌿
Git
Предустановлен на большинстве Linux-дистрибутивов. На macOS устанавливается с Xcode Command Line Tools.
02
Установка Node.js 22+
Через nvm — рекомендуемый способ для обеих систем

Установка nvm (Linux и macOS)

bash 
$ curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash

# Применить изменения в текущей сессии
$ source ~/.bashrc   # bash
$ source ~/.zshrc    # zsh (macOS по умолчанию)

# Установить Node.js 22
$ nvm install 22
$ nvm use 22
$ node --version    # должно показать v22.x.x
🐧 Linux — альтернатива
bash — Ubuntu/Debian
$ curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
$ sudo apt-get install -y nodejs
🍎 macOS — альтернатива
bash — Homebrew
$ brew install node@22
$ brew link node@22
03
Установка Claude Code CLI
Глобальная установка — обязательна для работы Agent SDK

Установка через npm

bash 
$ npm install -g @anthropic-ai/claude-code

Или нативный инсталлятор (рекомендуется)

Нативный инсталлятор создаёт настоящий бинарник — Agent SDK найдёт его без дополнительной настройки:

bash 
# Сначала установить через npm, затем выполнить нативную установку
$ npm install -g @anthropic-ai/claude-code
$ claude install

# Добавить в PATH (если не добавлено автоматически)
$ echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc
$ source ~/.bashrc

Проверить установку

bash 
$ which claude
$ claude --version
# Ожидаемый вывод: 2.x.x (Claude Code)
Нативный бинарник: После claude install исполняемый файл находится в ~/.local/bin/claude. Agent SDK находит его автоматически без дополнительной настройки.

07
Конфигурация .env
Настройка переменных окружения
bash 
$ cp .env.example .env
ПеременнаяПо умолчаниюОписание
ANTHROPIC_API_KEY — Оставить пустым если используется claude login
PORT 3009 Порт API-сервера
WEB_PORT 5180 Порт Web UI
AGENT_BYPASS_PERMISSIONS true Разрешить агенту выполнять команды без подтверждения
AGENT_USE_SUBAGENTS true false для ускорения (однопроходный режим)
POLL_INTERVAL_MS 30000 Интервал опроса задач координатором (мс)
DATABASE_URL ./data/aif.sqlite Путь к SQLite-файлу
LOG_LEVEL debug Уровень логов: fatal / error / warn / info / debug

Минимальный .env

.env 
# Авторизация (оставить пустым если claude login)
# ANTHROPIC_API_KEY=sk-ant-xxxxx

PORT=3009
WEB_PORT=5180
DATABASE_URL=./data/aif.sqlite

AGENT_BYPASS_PERMISSIONS=true
POLL_INTERVAL_MS=30000
LOG_LEVEL=debug
08
Авторизация Claude
Настроить доступ агента к Claude

Option A — Claude подписка (OAuth)

bash 
$ claude login

Откройте URL из терминала в браузере и завершите OAuth. Credentials сохранятся в ~/.claude/.credentials.json.

Option B — API Key

Добавьте в .env:

.env 
ANTHROPIC_API_KEY=sk-ant-xxxxx

Проверить авторизацию

bash 
$ ls ~/.claude/
# Должен быть файл .credentials.json

$ test -f ~/.claude/.credentials.json && echo "OK" || echo "NOT FOUND"

11
Устранение проблем
Типичные проблемы на Linux и macOS
!
claude: command not found после установки
Добавьте ~/.local/bin в PATH:
bash
$ echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc
$ source ~/.bashrc
# Для zsh (macOS):
$ echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc
$ source ~/.zshrc
!
EACCES — нет прав при npm install -g
Не используйте sudo с npm. Настройте глобальную директорию npm в домашней папке:
bash
$ mkdir -p ~/.npm-global
$ npm config set prefix '~/.npm-global'
$ echo 'export PATH="$HOME/.npm-global/bin:$PATH"' >> ~/.bashrc
$ source ~/.bashrc
!
Агент зависает или часто таймаутится
Добавьте в .env:
.env
AGENT_USE_SUBAGENTS=false
Перезапустите npm run dev.
!
Порт уже занят (EADDRINUSE)
Найдите и завершите процесс или измените порты в .env:
bash
# Найти что занимает порт
$ lsof -i :3009
$ lsof -i :5180
!
macOS — Gatekeeper блокирует claude
Если macOS блокирует запуск claude:
bash — macOS
$ xattr -d com.apple.quarantine ~/.local/bin/claude
